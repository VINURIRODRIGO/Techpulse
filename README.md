## General Information
Parkit Sign in, Sign out, Forgot Password and conformation emails using AWS Amplify.

## Technologies Used
Project is created with:
* Node.js version: 14.8.1
* git version: 2.32.0
* npm version: 6.14.15

## Setup
Install npm package.
```
npm install
```
To run the sign in and sign out pages.
```
npm start 
npm run android
```
